# md-to-pdf

https://github.com/simonhaenisch/md-to-pdf/blob/master/readme.md

## Install

```sh
npm install -g md-to-pdf
npx puppeteer browsers install chrome
```

## Run

```sh
./mdpdf.sh vuejs
```

Takes content/vuejs.md and generates output/vuejs.pdf

## Watch mode

```sh
./mdpdf.sh vuejs --watch
```
